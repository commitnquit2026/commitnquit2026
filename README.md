### `git commit -m "why we exist"`

We are **Commit & Quit** — a team that **commits** everything to the hackathon and **quits** only after shipping something legendary.

> *"We don't do half measures. We push to production, then we sleep."*

**Our Philosophy:**
- 🟢 `commit` → Give 100% to every line of code
- ⚡ `push` → Ship fast, ship fearlessly  
- 🔴 `quit` → Walk away only when it's done right
- 🏆 `merge` → Combine skills to beat the best

</td>
<td width="50%">

```python
class CommitAndQuit:
    def __init__(self):
        self.team     = "Commit & Quit"
        self.event    = "ODOO × INDUS Hackathon"
        self.stack    = ["Odoo", "Python", "JS"]
        self.attitude = "Ship or die trying 🔥"
        self.sleep    = False  # not during hackathon
    
    def execute(self):
        while self.hackathon_live():
            self.code()
            self.commit()
            self.push()
        return "🏆 Champion"

team = CommitAndQuit()
team.execute()
```
